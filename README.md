**北京交通大学学生违纪处分办法**

**第二章 违纪行为与处分**

**第二十四条 学生应当诚实守信，遵守学术规范，在学业、科研、品行等方面有严重失信行为的，给予下列处分：**

（一）在评奖评优、奖助学金申请、就业推荐、学籍与学位管理等活动中提供虚假材料或者隐瞒重要事实，情节较轻的，给予警告或者严重警告处分；情节较重或者经教育不改的，给予记过以上处分。

（二）在科研或课程学习过程中伪造、篡改实验数据、实验记录、图表、文献引用等，或者捏造研究成果的，视情节轻重，给予严重警告以上处分；情节严重的，可以给予留校察看或者开除学籍处分。

**（三）剽窃、抄袭、侵占他人研究成果或者学术观点等违反学术诚信的行为，经学校学术道德相关机构认定后，视情节轻重，给予记过以上处分；情节严重的，可以给予留校察看或者开除学籍处分。**

（四）学位论文、课程论文或者公开发表的研究成果存在抄袭、篡改、伪造等学术不端行为，或者参与代写、买卖论文等行为的，经学校学术道德委员会、学位评定机构等认定后，给予开除学籍处分；已获得的学位、荣誉或者奖励，依规予以撤销并向社会通报。
